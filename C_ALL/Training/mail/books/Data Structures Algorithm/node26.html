Destructors

None of the above examples defines a ``destructor'' for a class. While constructor functions are invoked whenever an object is created (e.g., using ``new'', or by declaring a variable of the given type) destructor functions are invoked whenever an object is destroyed (using delete, or when a function exits and its local variables go out of scope). Destructors are only essential when data fields involve pointers (dynamic data). Then we must have a destructor if we want to delete pointed to data when an object is deleted (this will be illustrated in later examples), In the above example we don't have any such data, so we don't need to define a destructor.
