Constructors

Whenever the data fields of an object include pointers we must normally have a constructor to allocate space for the pointed to data (unless the object's datafields will always point to pre-existing objects). This was illustrated in the last example: 

    namedstack::namedstack(char* name)
    {
        stkname = new char[strlen(name) + 1];
        strcpy(stkname, name);
    }

Linked list examples from last term should also illustrate the use of constructors for objects based on dynamic data.
