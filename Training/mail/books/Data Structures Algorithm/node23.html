Derived classes:

A more specific class such as `square' is referred to as a derived class, and the more general class it is derived from is referred to as the base class (e.g., `shape'). If a class declaration begins with: 

class DerivedClass : BaseClass {

This means that BaseClass is a private base class of DerivedClass, and the derived class can't invoke the base classes member functions. But usually we DO want derived classes to access base classes functions, so what we want is: 

class DerivedClass : public BaseClass {
